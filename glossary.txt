USER
	A user is a person who uses with the system
MEMBER
	A member is a user that is registered to the system
GUEST
	A user who is not a member
MODERATOR
	A moderator is a member with rights to share,close and invite new persons to an existing conference
ADMINISTRATOR
	An administrator is a member with rights to configurate higher settings
PLUGIN
	A plugin is an application provided from the system
CONFERENCE
	A conference is an instantiation of at least one plugin
DATABASE
	A database is a structured collection of member data.
